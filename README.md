# 📖 Project Description

**Find My Sport** is a ReactJS web application that solves a common problem: people who want to start a sport but do not know which one suits them or where to begin.

Users answer three quick questions about their training preferences, and the application returns a personalised ranked list of sports. Each sport links to a dedicated venue page showing three real Lebanese venues with addresses and registration phone numbers.

This project represents **Phase 2** of the coursework, migrating the original HTML/CSS website from Phase 1 into a modern ReactJS application with:

* Dynamic quiz logic
* Component-based architecture
* Responsive UI design
* State management using React Hooks
* Automated unit testing

### Covered Sports

* Football ⚽
* Basketball 🏀
* Swimming 🏊
* Tennis 🎾

### Venue Coverage

Each sport includes 3 real venues located across Lebanon:

* Beirut
* Mount Lebanon
* North Lebanon

# ✨ Features

| Feature              | Description                                                               |
| -------------------- | ------------------------------------------------------------------------- |
| 🧠 Smart Quiz        | 3-question quiz with weighted scoring algorithm and automatic progression |
| 🏆 Ranked Results    | Sports sorted by compatibility score with highlighted “Best Match”        |
| 📍 Venue Directory   | 12 real Lebanese venues with addresses and contact numbers                |
| 🎨 Sport Theming     | Dynamic color themes using CSS custom properties                          |
| 📱 Responsive Design | Mobile-friendly layout using CSS Grid and Flexbox                         |
| ✅ Unit Testing       | 22 tests covering logic, rendering, and data validation                   |
| 📬 Contact Form      | Validated form with success confirmation state                            |
